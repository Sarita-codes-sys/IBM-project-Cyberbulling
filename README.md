IBM PROJECT-Cyberbulling
Cyberbullying Detection System

Overview

The Cyberbullying Detection System is a web-based application developed to identify and analyze cyberbullying content in text messages and social media posts. The project uses Machine Learning and Natural Language Processing (NLP) techniques to classify text as either cyberbullying or non-cyberbullying, helping create a safer online environment.

Features

- Real-time cyberbullying text detection
- User-friendly web interface
- Machine Learning-based classification
- Fast and accurate predictions
- Responsive design for desktop and mobile devices

Technologies Used

- Python
- Flask
- HTML
- CSS
- Machine Learning
- Natural Language Processing (NLP)

Project Objective

The main objective of this project is to detect harmful and abusive online content automatically, reducing cyberbullying and promoting responsible digital communication.

Future Enhancements

- Multi-language support
- Advanced deep learning models
- Social media integration
- Sentiment analysis dashboard
